# Flask Stock Tracker Dashboard

A modern Flask-based web dashboard for displaying stock market analysis data from the `Top_Stocks.py` Python script. Features interactive tabs for top global stocks and watchlist monitoring with OHLC charts and price data.

## Features

✨ **Modern Web Dashboard**
- Responsive design that works on desktop, tablet, and mobile
- Dark theme with professional styling
- Real-time chart rendering with Chart.js

📊 **Two Main Views**
1. **Top Global Stocks** - Top 5 movers with percentage changes
2. **Watchlist** - Monitored stocks with detailed analytics

📈 **Interactive Charts**
- 7-day OHLC (Open, High, Low, Close) line charts
- Symbol-switching buttons to view different stocks
- Detailed price tables for each day

📋 **Price Analytics**
- 7-day high/low statistics
- Current price and average price calculations
- Color-coded positive (green) / negative (red) performance

## Installation & Setup

### Step 1: Install Flask Dependencies

```bash
cd flask_app
pip install -r requirements.txt
```

Or manually install:
```bash
pip install Flask==2.3.3
```

### Step 2: Generate Stock Data

Run the Python script to generate stock data JSON:

```bash
cd ..
python Top_Stocks.py
```

This creates `flask_app/data/stock_data.json` with:
- Top 5 global movers
- Top movers OHLC data
- Watchlist performance
- Watchlist OHLC data

### Step 3: Start Flask Server

```bash
cd flask_app
python app.py
```

The dashboard will be available at: **http://localhost:5000**

## Usage

### Main Dashboard
- **Top Global Stocks Tab**: View the 5 highest and lowest performing stocks
- **Watchlist Tab**: Monitor your custom watchlist

### Switching Symbols
Click any symbol button to switch between stocks and view their:
- 7-day line charts (Open, High, Low, Close)
- Detailed price tables
- Price statistics (high, low, current, average)

### Refreshing Data
Click the **🔄 Refresh Data** button in the header to reload data from the JSON file.

### Chart Interaction
- Hover over chart points to see exact prices
- Charts are fully responsive and resize for mobile devices

## Data Flow

```
Top_Stocks.py (Python Script)
    ↓
    Generates stock data with OHLC information
    ↓
flask_app/data/stock_data.json
    ↓
Flask App (app.py)
    ↓
    Reads JSON and serves via templates
    ↓
Dashboard (index.html + JavaScript)
    ↓
    Displays charts, tables, and statistics
```

## API Endpoints

### GET `/`
Returns the main dashboard HTML

**Response**: HTML dashboard page

### GET `/api/stock-data`
Returns all stock data as JSON

**Response**:
```json
{
  "timestamp": "2024-05-04T12:34:56.789123",
  "topMovers": {
    "symbols": ["NVDA", "TSLA", ...],
    "performance": [["NVDA", 4.85], ...],
    "ohlc": {
      "NVDA": {
        "dates": ["2024-04-29", ...],
        "opens": [123.45, ...],
        "highs": [125.89, ...],
        "lows": [122.34, ...],
        "closes": [124.78, ...]
      }
    }
  },
  "watchlist": { ... },
  "symbolNames": { "AAPL": "Apple Inc.", ... }
}
```

### GET `/api/refresh`
Reloads stock data from JSON file

**Response**:
```json
{
  "success": true,
  "timestamp": "2024-05-04T12:34:56.789123",
  "message": "Data refreshed successfully"
}
```

## Customization

### Colors
Edit CSS variables in `static/css/style.css`:

```css
:root {
    --primary-color: #3b82f6;      /* Primary blue */
    --success-color: #22c55e;      /* Green for gains */
    --danger-color: #ef4444;       /* Red for losses */
    --warning-color: #f59e0b;      /* Orange */
    --bg-dark: #0f172a;            /* Dark background */
}
```

### Watchlist Symbols
Edit `Top_Stocks.py` to change the watchlist:

```python
WATCHLIST = ["AAPL", "NVDA", "MSFT", "META", "GOOGL"]  # Modify this list
```

### Symbol Names
Add more symbols and company names in `Top_Stocks.py`:

```python
SYMBOL_NAMES = {
    "AAPL": "Apple Inc.",
    "MSFT": "Microsoft Corporation",
    # Add more symbols here...
}
```

### Chart Appearance
Modify chart options in `static/js/script.js`:

```javascript
const chartOptions = {
    responsive: true,
    maintainAspectRatio: true,
    // Customize chart rendering...
};
```

## Troubleshooting

### "No stock data available" Error
**Solution**: Run the Python script first
```bash
python Top_Stocks.py
```

### Port 5000 Already in Use
**Solution**: Change the port in `app.py`:
```python
app.run(debug=True, port=5001)  # Use a different port
```

### Blank Dashboard or No Charts
**Possible causes**:
1. Stock data JSON not generated (run `Top_Stocks.py`)
2. JavaScript error - check browser console (F12)
3. Chart.js not loading - check internet connection

### Charts Not Responsive
**Solution**: Add this to `app.py`:
```python
app.config['SEND_FILE_MAX_AGE_DEFAULT'] = 0
```

## Performance Tips

1. **Minimize JSON File Size**: Remove old data periodically
2. **Cache Data**: Implement caching for less frequent updates
3. **Lazy Load Charts**: Charts are created on-demand when symbols are selected
4. **Compress Assets**: Use CSS/JS minification for production

## Deployment

### Deploy to Heroku

1. Create `Procfile`:
```
web: python flask_app/app.py
```

2. Create `runtime.txt`:
```
python-3.10.12
```

3. Deploy:
```bash
git add .
git commit -m "Deploy to Heroku"
git push heroku main
```

### Deploy to PythonAnywhere

1. Upload files
2. Create Flask web app
3. Set working directory to project root
4. Update WSGI file to import from `flask_app.app`
