Connection Tests

Configuration Files

.env.local (should use /api for proxy)

REACT_APP_API_URL=/api

setupProxy.js (should exist in src/)

const { createProxyMiddleware } = require('http-proxy-middleware'); module.exports = function(app) { app.use( '/api', createProxyMiddleware({ target: 'http://localhost:5000', changeOrigin: true, }) ); };

package.json (must have http-proxy-middleware)

"dependencies": { ... "http-proxy-middleware": "^2.0.6" }

Steps to Fix

  1. Install missing package:
    npm install
  2. Stop npm start (if running)
    ⚠️ You must restart npm start after installing the package
  3. Start fresh:
    npm start

    npm start will restart in dev mode with proxy enabled

  4. Verify API is connected:

    Status badge should show "● API Connected" (green)

Troubleshooting

If Still Disconnected After Steps Above:

✓ Make sure backend is running:
cd backend_v2
python main.py

Should see: Uvicorn running on http://0.0.0.0:5000

✓ Open DevTools and check Console:
  1. Press F12 to open DevTools
  2. Go to Console tab
  3. Look for error messages like:
    • "Failed to fetch" = Backend not running or CORS blocked
    • "Request timeout" = Backend running but slow
    • "Cannot find module" = Missing http-proxy-middleware
✓ Check Network tab:
  1. Open DevTools → Network tab
  2. Refresh the page
  3. Look for request to "health"
  4. Check Status code: should be 200
  5. Check Preview: should show {"status":"ok","message":"Stock API is running"}

Common Errors & Solutions

Error: Cannot find module 'http-proxy-middleware'

Solution: Run npm install then restart npm start

Error: CORS policy blocked the request

Solution: Using proxy bypasses CORS. Make sure .env.local has REACT_APP_API_URL=/api

Error: Backend not responding

Solution: Ensure backend is running on port 5000: python backend_v2/main.py

What Should Happen

  1. Frontend starts on localhost:3000
  2. Backend starts on localhost:5000
  3. setupProxy.js intercepts /api requests
  4. Requests are routed to http://localhost:5000/api
  5. No CORS errors
  6. Status shows "● API Connected" (green)
  7. Stock data loads and displays
